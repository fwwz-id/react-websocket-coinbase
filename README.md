# react-websocket-coinbase
A simple React, WebSocket, and Coinbase API

## Live preview
http://react-websocket-coinbase.vercel.app/

## Clone to your local directory
```
git clone https://github.com/fwwz-id/react-websocket-coinbase.git
```

## Scripts
### Install
```
yarn
```
### Development
```
yarn dev
```
### Build
```
yarn build
```
